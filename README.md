# ChatGPT-Reviews-Analysis
Sentiment Analysis of ChatGPT Customer Reviews using Python

## Project Overview
This project analyzes customer reviews of ChatGPT using Python and NLP techniques.

## Objectives
- Analyze customer sentiment
- Calculate polarity and subjectivity
- Extract keywords
- Generate visualizations

## Tools Used
- Python
- Pandas
- TextBlob
- Matplotlib
- Seaborn
- WordCloud

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Sentiment Analysis
4. Keyword Extraction
5. Visualization

## Key Findings
- Most reviews are positive.
- Ratings align strongly with sentiment.
- Users appreciate helpfulness and accuracy.
- Negative reviews mainly concern limitations and inaccuracies.

## Conclusion
The project demonstrates how NLP can transform customer feedback into actionable business insights.
